Connect your Google Sheet

This dashboard reads and writes live data through a Google Apps Script web app. One-time setup takes about 5 minutes.

  1. Open your finance Google Sheet → Extensions → Apps Script
  2. Delete any existing code and paste the full contents of apps_script.gs
  3. Deploy → New deployment → type Web app, execute as Me, access Anyone → Deploy & authorize
  4. Copy the Web app URL and paste it below
